Tema 2: Elementos que Componen la Estructuración de una Página Web

  • 1. Contenedor (wrapper)

    El contenedor es el elemento principal que agrupa todo el contenido de la página. Ayuda a centrar y establecer un ancho fijo o máximo para la página.

  • 2. Cabecera (header)

    La cabecera generalmente contiene el logotipo del sitio, el título, y elementos de navegación como el menú principal.

  • 3. Contenido (content)

    La sección de contenido es donde se ubica el contenido principal de la página, como texto, imágenes, y otros elementos multimedia.

  • 4. Menú (menu)

    El menú proporciona enlaces de navegación que permiten a los usuarios desplazarse a otras partes del sitio web.

  • 5. Pie (footer)

    El pie de página suele contener información de contacto, enlaces a políticas de privacidad, y otros enlaces útiles.

  • 6. Lateral (sidebar)

    El lateral es una sección de la página que puede contener enlaces adicionales, anuncios, o cualquier otro contenido complementario.

  • 7. Etiqueta div

    La etiqueta <div> es un contenedor genérico utilizado para agrupar elementos y aplicar estilos o scripts a un grupo específico de contenido.

Tema 3: Estructuración Semántica de una Página Web

  • 1. <header>

    Elemento que representa el encabezado de una sección o página, normalmente contiene logotipos, títulos, y menús de navegación.

  • 2. <nav>

    Elemento utilizado para contener enlaces de navegación.

  • 3. <main>

    Elemento que representa el contenido principal de un documento, debe ser único y no estar anidado dentro de otros elementos de contenido.

  • 4. <article>

    Elemento utilizado para contenido que puede ser distribuido de forma independiente, como artículos, publicaciones de blog, etc.

  • 5. <section>

    Elemento que define secciones dentro de un documento, cada una con su propio encabezado.

  • 6. <aside>

    Elemento que representa contenido secundario relacionado con el contenido principal, como barras laterales o bloques de anuncios.

  • 7. <footer>

    Elemento que define el pie de página de una sección o documento, generalmente contiene información de contacto y derechos de autor.

  • 8. <h1> a <h6>

    Elementos que definen los encabezados de diferentes niveles, donde <h1> es el encabezado de nivel más alto y <h6> el más bajo.

  • 9. <p>

    Elemento que define un párrafo de texto.

  • 10. <address>

    Elemento que proporciona información de contacto para su sección o su dueño.

  • 11. <time>

    Elemento que representa una fecha o una hora específica, o ambos.
